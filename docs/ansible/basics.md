# Core Concepts

1. **Inventory**  
   Hosts and groups managed by Ansible.

2. **Playbooks**  
   YAML files defining automation workflows.

3. **Modules**  
   Units of work (packages, files, services).

4. **Roles**  
   Structured, reusable playbook components.
