# Installing Ansible

#Using pip

```bash
pip install ansible
```

#Ubuntu / Debian

```bash
sudo apt update
sudo apt install ansible
```

#macOS (Homebrew)

```bash
brew install ansible
```

#Windows

Use Ansible inside **WSL (Ubuntu/Debian)**.

#Verify Installation

```bash
ansible --version
```

!!! warning
Do not mix pip-based installs with OS package installs.
